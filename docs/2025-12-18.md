# Rotina de Desenvolvimento - 18/12/2025

**Timestamp:** 2025-12-18 (criado durante sessão de desenvolvimento)

---

## Planejado

### Expansão Multi-Crypto
- Expandir de 3 para 26+ criptomoedas
- Otimizar uso da API CoinGecko (tier gratuito)
- Gráficos sob demanda (economia de requisições)

---

## Implementado

### 1. Expansão de Moedas (26 criptomoedas)

**Arquivo:** `lib/utils/config.dart`

Moedas adicionadas:
- **Top 10:** BTC, ETH, XRP, SOL, ADA
- **Layer 1 & 2:** AVAX, DOT, NEAR, ATOM, ALGO
- **DeFi:** LINK, UNI, AAVE, GRT, LDO
- **Layer 2 Scaling:** POL, ARB, OP, IMX
- **Outros:** XLM, HBAR, BCH, LTC, ETC, QNT, TIA

Cada moeda com:
- ID CoinGecko
- Nome completo
- Símbolo
- Cor da marca

### 2. Nova Arquitetura de UI

**Home Screen:** Lista compacta sem gráficos
- Economia de ~26 requisições ao iniciar
- 1 request para preços de todas as moedas (a cada 60s)

**Detail Screen:** `lib/ui/screens/crypto_detail_screen.dart`
- Gráfico carregado sob demanda ao abrir
- Seletor de período (24H, 7D, 1M, 1A)
- Cache local por período
- Simulador de investimento integrado

### 3. Remoções
- Removido Tesouro Selic (fora do escopo de criptos)
- Simplificado controller (sem carregamento automático de históricos)

### 4. Otimização de API

| Antes | Depois |
|-------|--------|
| 3 moedas | 26 moedas |
| ~4 req/min | ~1 req/min |
| Gráficos sempre | Gráficos sob demanda |
